## Welcome to ProjetosVoluntariosTech

ProjetosVoluntariosTech uses technology and volunteer skills to create digital solutions for local communities. Our projects include website templates for microentrepreneurs, religious organizations, and community initiatives - all built with beginner-friendly HTML, CSS, and JavaScript.

<Note>
All projects are designed to be accessible for contributors of any technical level. No complex frameworks or build tools required!
</Note>

## Getting Started in 3 Steps

<Steps>
  <Step title="Clone the Repository">
    First, download the project repository to your local machine:

    ```bash
    git clone https://github.com/Yudikom/ProjetosVoluntariosTech.git
    cd ProjetosVoluntariosTech
    ```

    Or download the ZIP file directly:
    - Visit the [GitHub repository](https://github.com/Yudikom/ProjetosVoluntariosTech)
    - Click the green "Code" button
    - Select "Download ZIP"
    - Extract the files to your preferred location
  </Step>

  <Step title="Explore the Project Structure">
    The repository is organized into individual project folders:

    ```
    ProjetosVoluntariosTech/
    ├── README.md
    ├── projeto1-Escola-Biblica/
    │   └── index.html
    └── projeto1-solucao-refrigeracao/
        └── index.html
    ```

    Each project is a self-contained website with:
    - **Single HTML file** - All code in one place for easy editing
    - **Inline CSS** - Styling embedded in `<style>` tags
    - **Vanilla JavaScript** - No external dependencies
    - **CDN resources** - External libraries loaded from CDNs (Tailwind, Font Awesome)
  </Step>

  <Step title="Open and Preview Projects">
    Simply open any `index.html` file in your web browser:

    **Windows:**
    ```bash
    start projeto1-Escola-Biblica/index.html
    ```

    **macOS:**
    ```bash
    open projeto1-Escola-Biblica/index.html
    ```

    **Linux:**
    ```bash
    xdg-open projeto1-Escola-Biblica/index.html
    ```

    Or simply double-click the HTML file in your file explorer!
  </Step>
</Steps>

## Project Deep Dive

### Escola Bíblica Project

A complete educational portal for Bible studies with gamification features.

**Key Features:**
- 📅 **Interactive Calendar** - Navigate through 13 lessons with arrow controls
- 📖 **Quiz System** - Embedded Google Forms for each lesson
- 🏆 **Ranking System** - Leaderboard for student engagement
- 🎧 **Podcast Integration** - Audio lessons via Google Drive links

**Code Example - Navigation System:**

```javascript
function move(s) { 
    currentIdx = (currentIdx + s + lessons.length) % lessons.length; 
    render(); 
}

function tab(id) { 
    document.querySelectorAll('.section').forEach(s => s.classList.remove('active'));
    document.querySelectorAll('.nav-btn').forEach(b => b.classList.remove('active'));
    document.getElementById(id).classList.add('active');
    document.getElementById('btn-' + id).classList.add('active');
    window.scrollTo(0,0);
}
```

**Code Example - Lesson Data Structure:**


### Solução Refrigeração Project

A modern, responsive business website for a refrigeration services company.

**Key Features:**
- 🎨 **Tailwind CSS** - Utility-first styling via CDN
- 📱 **Responsive Design** - Mobile-first approach
- 💬 **WhatsApp Integration** - Floating button with pulse animation
- 🌐 **Service Showcase** - Comprehensive service listings with Font Awesome icons

**Code Example - WhatsApp Float Button:**

```html
<a href="https://wa.me/5564992902925?text=Olá,%20gostaria%20de%20fazer%20um%20orçamento" 
   class="whatsapp-float bg-green-500 text-white rounded-full p-4 shadow-lg" 
   target="_blank">
    <i class="fab fa-whatsapp text-4xl"></i>
</a>
```

```css
.whatsapp-float {
    position: fixed;
    bottom: 20px;
    right: 20px;
    z-index: 1000;
    animation: pulse 2s infinite;
}

@keyframes pulse {
    0% { transform: scale(1); box-shadow: 0 0 0 0 rgba(37, 211, 102, 0.7); }
    70% { transform: scale(1.05); box-shadow: 0 0 0 15px rgba(37, 211, 102, 0); }
    100% { transform: scale(1); box-shadow: 0 0 0 0 rgba(37, 211, 102, 0); }
}
```

**Code Example - Hero Section with Gradient Effects:**


## Making Your First Contribution

<Steps>
  <Step title="Choose a Project to Improve">
    Browse the existing projects and identify an area for improvement:
    - Add new features
    - Improve accessibility
    - Enhance mobile responsiveness
    - Fix bugs or issues
    - Update content
  </Step>

  <Step title="Edit the HTML File">
    Open the `index.html` file in your favorite code editor:

    ```bash
    # Using VS Code
    code projeto1-Escola-Biblica/index.html

    # Using nano
    nano projeto1-Escola-Biblica/index.html
    ```

    Make your changes and save the file.
  </Step>

  <Step title="Test Your Changes">
    Refresh the browser to see your changes in real-time:

    <CodeGroup>
    ```bash Windows
    # No build step needed - just refresh your browser!
    # Press Ctrl+R or F5
    ```

    ```bash macOS
    # No build step needed - just refresh your browser!
    # Press Cmd+R
    ```
    </CodeGroup>
  </Step>

  <Step title="Share Your Contribution">
    Once you're happy with your changes, share them with the community:

    ```bash
    git add .
    git commit -m "Improved accessibility in Escola Bíblica navigation"
    git push origin main
    ```
  </Step>
</Steps>

<Warning>
Before pushing changes, make sure all links are working and the page displays correctly on both desktop and mobile devices.
</Warning>

## Development Tips

### Using Browser DevTools

All projects can be debugged using your browser's developer tools:

1. **Right-click** on any element → **Inspect**
2. Edit CSS in real-time in the **Styles** panel
3. Debug JavaScript in the **Console** tab
4. Test responsive design in **Device Mode** (Ctrl+Shift+M)

### Common Customizations

<Accordion title="Changing Colors">
  Both projects use CSS custom properties for easy theming:

  ```css
  :root { 
      --p: #1a365d;  /* Primary color */
      --s: #2b6cb0;  /* Secondary color */
      --a: #ed8936;  /* Accent color */
      --bg: #f7fafc; /* Background */
  }
  ```

  Simply update these values to rebrand the entire site!
</Accordion>

<Accordion title="Adding New Lessons (Escola Bíblica)">
  Add a new object to the `lessons` array:

  ```javascript
  { 
      id: 14, 
      t: "LIÇÃO 14 – Your New Lesson Title", 
      d: "03/04/2026", 
      prof: "Your Name", 
      img: "https://your-image-url.jpg", 
      status: "Em breve", 
      desc: "Lesson description", 
      prayer: "Prayer topic", 
      pod: "#", 
      form: null 
  }
  ```
</Accordion>

<Accordion title="Adding Services (Solução Refrigeração)">

  ```html
  </div>
  ```
</Accordion>
