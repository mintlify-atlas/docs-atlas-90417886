## Project Template Approach

All projects in this repository follow a consistent structure:

- **Single-file architecture**: Each project is a self-contained HTML file
- **Inline styling**: CSS embedded directly for easy deployment
- **Vanilla JavaScript**: No external dependencies for core functionality
- **Mobile-first**: Responsive design for all screen sizes

## Community Impact

These projects serve different community needs:

1. **Educational**: The Bible School portal facilitates religious education with interactive features
2. **Commercial**: The refrigeration website helps local businesses reach customers online
3. **Accessible**: Both projects are optimized for low-bandwidth environments

<Info>
  All projects are volunteer contributions designed to be easily maintained and deployed without requiring complex infrastructure.
</Info>

## Technology Stack

The projects demonstrate proficiency in:

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Custom properties, flexbox, grid, and animations
- **JavaScript ES6+**: Modern syntax, DOM manipulation, and state management
- **Tailwind CSS**: Utility-first CSS framework (Solução project)
- **Responsive Design**: Mobile-first approach with media queries
