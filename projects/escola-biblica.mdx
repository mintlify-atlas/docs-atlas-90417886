## Overview

The Escola Bíblica (Bible School) portal is an interactive web application designed to facilitate religious education. It features a lesson calendar, embedded quiz system, and student ranking functionality.

<Note>
  This project demonstrates advanced DOM manipulation, state management, and dynamic content rendering using vanilla JavaScript.
</Note>

### Tab Switching Logic

```javascript
function tab(id) { 
  document.querySelectorAll('.section').forEach(s => s.classList.remove('active'));
  document.querySelectorAll('.nav-btn').forEach(b => b.classList.remove('active'));
  document.getElementById(id).classList.add('active');
  document.getElementById('btn-' + id).classList.add('active');
  window.scrollTo(0,0);
}
```

## Carousel System

### Lesson Card Rendering

The carousel dynamically renders lesson cards with status-based styling:

```javascript
function render() {
  const l = lessons[currentIdx];
  const statusClass = l.status === "Aula Ministrada" ? "status-done" : "status-soon";
  const podCall = l.status === "Aula Ministrada" 
    ? "Reveja o que aprendemos!" 
    : "Se prepare para essa aprendizagem!";

  document.getElementById('card-content').innerHTML = `
    <img src="${l.img}" class="lesson-img">
    <div class="lesson-content">
      <span class="status-badge ${statusClass}">${l.status}</span>
      <div style="font-size:0.9rem; color:var(--a); font-weight:bold; margin-bottom:5px;">
        📅 ${l.d} | Prof: ${l.prof}
      </div>
      <h2 class="lesson-title">${l.t}</h2>
      <div class="info-box">
        <strong>📖 Descrição:</strong><br>
        <span style="font-size:0.95rem; color:#4a5568;">${l.desc}</span>
      </div>
      <div class="prayer-box">
        <strong>🙏 Propósito de Oração desta lição:</strong><br>
        <span style="font-size:0.95rem;">${l.prayer}</span>
      </div>
      <div class="podcast-call">
        <p style="margin:0; color:#744210; font-weight:bold;">• ${podCall}</p>
        <a href="${l.pod}" target="_blank" class="btn-podcast">🎧 ESCUTAR PODCAST</a>
      </div>
    </div>
  `;
}
```

### Navigation Controls

```javascript
let currentIdx = 0;

function move(s) { 
  currentIdx = (currentIdx + s + lessons.length) % lessons.length; 
  render(); 
}
```

<Info>
  The modulo arithmetic `(currentIdx + s + lessons.length) % lessons.length` ensures circular navigation, allowing users to loop from the last lesson back to the first.
</Info>

## Quiz Integration

### Dynamic Quiz Selector

Bullet-style navigation for selecting quizzes:

```javascript
document.getElementById('quiz-bullets').innerHTML = lessons.map((item, i) => `
  <div class="quiz-bullet ${i === quizIdx ? 'active' : ''}" onclick="setQuiz(${i})">
    Lição ${item.id}
  </div>
`).join('');
```

### Embedded Google Forms

```javascript
const q = lessons[quizIdx];
const qContainer = document.getElementById('quiz-container');

if (q.form) {
  qContainer.innerHTML = `<iframe src="${q.form}">Carregando…</iframe>`;
} else {
  qContainer.innerHTML = `
    <div style="text-align:center; padding:40px; border:2px dashed #ccc; 
                border-radius:15px; background:white;">
      <h3>📖 Lição ${q.id}</h3>
      <p>Questionário liberado em breve!</p>
    </div>
  `;
}
```
