## Hacktoberfest

### What is Hacktoberfest?

Hacktoberfest is an annual celebration of open source software that takes place every October. It's a month-long event where contributors can:

- Make meaningful contributions to open source projects
- Learn about collaboration and version control
- Earn digital rewards and recognition
- Connect with the global open source community

### Our Participation

ProjetosVoluntariosTech actively participates in Hacktoberfest! We prepare special issues, provide mentorship, and create a welcoming environment for first-time contributors.

<Tip>
Hacktoberfest is an excellent time to make your first contribution! We label beginner-friendly issues and provide extra support during this period.
</Tip>

**What we offer during Hacktoberfest:**

- **Curated Issues**: Specially labeled tasks suitable for various skill levels
- **Enhanced Support**: More maintainers available to review PRs and answer questions
- **Documentation Tasks**: Non-code contributions for those new to programming
- **Community Recognition**: Special acknowledgment for Hacktoberfest contributors

### How to Participate

1. **Register** at the official Hacktoberfest website
2. **Browse our repositories** for issues tagged with `hacktoberfest`
3. **Submit quality pull requests** between October 1-31
4. **Engage with reviewers** and iterate on your contributions
5. **Celebrate** your contributions with the community!

## Learning Sessions & Workshops

We regularly host educational events to help community members grow their skills and confidence.

### Topics We Cover

- **Getting Started with Git & GitHub**: Version control basics for beginners
- **Contributing to Open Source**: Best practices and common workflows
- **Educational Technology Tools**: Platforms and resources for community education
- **Web Development Basics**: HTML, CSS, JavaScript fundamentals
- **Design for Impact**: Creating user-friendly interfaces for community projects
- **Project Management**: Organizing and leading volunteer tech initiatives

<Note>
Workshop recordings and materials are often shared on our GitHub repository for those who can't attend live sessions.
</Note>

### Workshop Format

Our workshops typically include:

- **Live Demonstrations**: Step-by-step walkthroughs of concepts
- **Hands-on Practice**: Time to apply what you're learning
- **Q&A Sessions**: Opportunity to ask questions and get help
- **Follow-up Resources**: Links to tutorials, documentation, and practice exercises

## Code Sprints

Code sprints are focused, collaborative sessions where we work together on specific projects or features.

**What happens during a sprint:**

- Team formation around specific goals
- Collaborative problem-solving and pair programming
- Intensive development in a supportive environment
- Progress reviews and celebrations of achievements

<Tip>
Code sprints are great opportunities to dive deep into a project and make significant progress in a short time.
</Tip>

## How to Propose an Event

Have an idea for a workshop, learning session, or community event? We'd love to hear it!

### Steps to Propose an Event

1. **Open an Issue** on our GitHub repository with the `event-proposal` label
2. **Include details**:
   - Event type and format
   - Proposed topic or focus
   - Target audience and skill level
   - Estimated duration
   - Any resources needed
3. **Discuss with the community** and refine the proposal
4. **Coordinate timing** with maintainers
5. **Promote the event** through our channels

## How to Host an Event

Interested in leading a workshop or organizing an event? Here's how:

### Requirements for Hosts

- **Passion for the topic**: Enthusiasm is more important than perfection
- **Clear objectives**: Know what participants should learn or achieve
- **Preparation**: Have materials and examples ready
- **Flexibility**: Be ready to adapt to participant needs
- **Welcoming attitude**: Create an inclusive, supportive atmosphere

### We Provide

- Help with event planning and structure
- Promotion and participant recruitment
- Technical support and infrastructure
- Co-hosts or moderators if needed
- Feedback and appreciation

<Tip>
First-time hosts are welcome! We'll support you throughout the process and help ensure your event is successful.
</Tip>
